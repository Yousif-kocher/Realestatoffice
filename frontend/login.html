How to Access Your Login Page

Once you have set up your Node.js server, you can access your login page using one of these URLs:

http://localhost:3000/login.html

If you've configured your server to serve the login page at the root path, use:

http://localhost:3000

Make sure your server is running before trying to access these URLs.
